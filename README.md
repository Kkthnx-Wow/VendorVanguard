# VendorVanguard
> **Modular automated selling and repairing.**

VendorVanguard is a World of Warcraft addon designed to streamline your vendor interactions. It automatically handles selling junk items and repairing your gear whenever you visit a capable merchant, letting you get back to the adventure faster.

Its modular design means you have complete control over which automated features are active.

## Features

* **Automated Repairs:** Automatically repairs your equipped gear when visiting a repair vendor. It will prioritize using Guild funds if available and permitted.
* **Junk Selling:** Automatically sells poor-quality (gray) items upon opening a vendor window.
* **Modular Toggle:** Easily enable or disable individual features via the settings menu or chat commands.
* **Lightweight:** Designed to have minimal impact on game performance.

## Installation

1.  Download the latest release zip file.
2.  Extract the contents of the zip file into your World of Warcraft `_retail_/Interface/AddOns` folder.
3.  Ensure the folder inside AddOns is named `VendorVanguard`.
4.  Launch World of Warcraft and enable the addon at the character selection screen.

*(Alternatively, install via your preferred addon manager if available).*

## Usage

Once installed, VendorVanguard works automatically. Just open a vendor window, and it will perform its tasks based on your current settings.

**Chat Commands:**

Type `/vv` or `/vendorvanguard` in-game to open the configuration options or see a list of available commands.

* `/vv config` - Opens the interface options menu.
* `/vv repair` - Toggles auto-repair on/off.
* `/vv sell` - Toggles auto-selling of grays on/off.
